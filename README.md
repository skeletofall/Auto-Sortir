# 📂 Auto File Sorter (Автосортировщик файлов)

Простой и эффективный скрипт на Python для автоматической организации папки **Загрузки** (Downloads). Скрипт сортирует файлы по категориям (Изображения, Документы, Архивы и т.д.) и перемещает их в соответствующие подпапки.

## ✨ Возможности

- 🖼️ **Автоматическая сортировка**: Распределяет файлы по типам (Images, Documents, Audio, Video, Code, Archives и др.).
- ⚙️ **Настраиваемые категории**: Легко добавляйте свои типы файлов через `config.json`.
- 📁 **Выбор папки**: Настройте любую папку для сортировки, не только Downloads.
- 🛡️ **Безопасность**: Не трогает скрытые файлы и системные папки.
- 🔄 **Обработка дубликатов**: Если файл с таким именем уже существует в папке назначения, к имени добавляется суффикс `_1`, `_2` и т.д.
- 🧹 **Очистка**: Удаляет пустые папки, оставшиеся после перемещения файлов (кроме основных категорий).
- 📝 **Логирование**: Все действия записываются в файл `auto_organizer.log` в целевой папке.

## 🚀 Установка и запуск

### Требования
- Python 3.6 или выше.
- Никаких внешних библиотек не требуется (используются только стандартные модули: `os`, `shutil`, `pathlib`, `logging`, `json`).

### Инструкция

1. **Склонируйте репозиторий** или скачайте файлы:
```bash
   git clone https://github.com/skeletofall/auto-file-sorter.git
   cd auto-sortir
```

2. **Создайте конфигурационный файл**:
```bash
   cp config.example.json config.json
```

3. **Настройте config.json** (опционально):
   - Измените `target_directory` на нужную папку (по умолчанию `~/Downloads`)
   - Добавьте свои категории и расширения файлов
   - Пример:
```json
{
  "target_directory": "~/Downloads",
  "categories": {
    "Images": [".jpg", ".png", ".gif"],
    "MyCustomCategory": [".custom", ".special"]
  }
}
```

4. **Запустите скрипт**:
```bash
   python autosortir
```

Скрипт автоматически прочитает настройки из `config.json` и начнет сортировку.

## ⚙️ Как это работает

1. Скрипт читает настройки из `config.json`.
2. Сканирует указанную папку (по умолчанию `~/Downloads`).
3. Определяет расширение каждого файла.
4. Создает папку категории (если она еще не существует).
5. Перемещает файл в соответствующую папку.
6. Если файл с таким именем уже есть, создает копию с уникальным именем (например, `photo_1.jpg`).
7. В конце удаляет пустые посторонние папки в корневой директории.

## 🎨 Настройка категорий

Вы можете легко добавлять свои категории в `config.json`:

```json
{
  "target_directory": "~/Downloads",
  "categories": {
    "Images": [".jpg", ".png"],
    "MyPhotos": [".raw", ".cr2", ".nef"],
    "WorkDocs": [".docx", ".xlsx"]
  }
}
```

**Советы:**
- Расширения должны начинаться с точки (`.jpg`, а не `jpg`)
- Регистр не важен (`.JPG` и `.jpg` обрабатываются одинаково)
- Файлы без категории попадут в папку `Other`
