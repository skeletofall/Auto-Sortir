# 📂 Auto File Sorter (Автосортировщик файлов)

Простой и эффективный скрипт на Python для автоматической организации папки **Загрузки** (Downloads). Скрипт сортирует файлы по категориям (Изображения, Документы, Архивы и т.д.) и перемещает их в соответствующие подпапки.

## ✨ Возможности

- 🖼️ **Автоматическая сортировка**: Распределяет файлы по типам (Images, Documents, Audio, Video, Code, Archives и др.).
- 🛡️ **Безопасность**: Не трогает скрытые файлы и системные папки.
- 🔄 **Обработка дубликатов**: Если файл с таким именем уже существует в папке назначения, к имени добавляется суффикс `_1`, `_2` и т.д.
- 🧹 **Очистка**: Удаляет пустые папки, оставшиеся после перемещения файлов (кроме основных категорий).
- 📝 **Логирование**: Все действия записываются в файл `auto_organizer.log` в папке Загрузки.

## 📋 Поддерживаемые категории

| Категория | Расширения |
| :--- | :--- |
| **Images** | `.jpg`, `.jpeg`, `.png`, `.gif`, `.bmp`, `.svg`, `.webp` |
| **Documents** | `.pdf`, `.doc`, `.docx`, `.txt`, `.rtf`, `.xls`, `.xlsx`, `.ppt`, `.pptx` |
| **Archives** | `.zip`, `.rar`, `.7z`, `.tar`, `.gz` |
| **Audio** | `.mp3`, `.wav`, `.flac`, `.aac` |
| **Video** | `.mp4`, `.avi`, `.mov`, `.mkv`, `.wmv` |
| **Code** | `.py`, `.js`, `.html`, `.css`, `.java`, `.cpp`, `.c`, `.php` |
| **Executables** | `.exe`, `.msi`, `.dmg`, `.deb` |
| **Torrents** | `.torrent` |
| **Other** | Все остальные файлы |

## 🚀 Установка и запуск

### Требования
- Python 3.6 или выше.
- Никаких внешних библиотек не требуется (используются только стандартные модули: `os`, `shutil`, `pathlib`, `logging`).

### Инструкция

1. **Склонируйте репозиторий** или скачайте файл `auto_sortir.py`:
```bash
   git clone https://github.com/ВАШ_НИК/auto-file-sorter.git
   cd auto-file-sorter
```

2. **Запустите скрипт**:
```bash
   python auto_sortir.py
```

3. Скрипт автоматически найдет папку **Downloads** в вашей домашней директории и начнет сортировку.

## ⚙️ Как это работает

1. Скрипт сканирует папку `~/Downloads`.
2. Определяет расширение каждого файла.
3. Создает папку категории (если она еще не существует).
4. Перемещает файл в соответствующую папку.
5. Если файл с таким именем уже есть, создает копию с уникальным именем (например, `photo_1.jpg`).
6. В конце удаляет пустые посторонние папки в корневой директории.
